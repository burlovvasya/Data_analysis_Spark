# Лабораторная работа №1: Анализ возвратов в E-commerce (Вариант 5)

## Описание
Анализ лога возвратов интернет-магазина с использованием PySpark

## Инструкция по запуску
1. Запустить Hadoop: `start-dfs.sh`, `start-yarn.sh`
2. Загрузить данные в HDFS: `hdfs dfs -put ecommerce_returns.csv /user/hadoop/lab_05/input/`
3. Запустить Jupyter: `jupyter notebook`
4. Выполнить все ячейки в `lab_05_returns_analysis.ipynb`

## Источник данных
Синтетический датасет, сгенерированный для лабораторной работы
